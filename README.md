# CV game
## Описание
CV game - это игра, в которой Вам нужно вставать в различные позы и набирать очки за наиболее точное повторение позы.

За каждую позу можно набрать от 0 до 100 баллов,
но фактически достичь рзультата в 100 баллов почти невозможно,
т.к. для этого углы наклона рук и ног должны идеально совпадать с эталонными, заложенными в программе.

В настройках игры можно указать количество раундов и длительность каждого раунда.
## Установка
Для установки программы вам нужно склонировать её репозиторий себе на компьютер и установить все зависимости.

Так как программа работает не на последней версии Python, а на Python 3.12, крайне рекомендуется создать виртуальное окрыжение с этой версией, и запускать программу в нём.

Пример установки программы на Windows с уже установленным Python 3.12:

    git clone https://github.com/GameKnight999999/cv_game.git
    cd cv_game
    py -3.12 -m venv .venv
    .venv\Scripts\pip install -r requirements.txt
## Запуск
Важно, чтобы во время игры в поле зрения камеры был **только один** человек и чтобы он был виден **в полный рост**.
Для запуска игры запустите `main.py`.
