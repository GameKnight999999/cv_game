# CV game
## Описание
CV game - это игра, в которой Вам нужно вставать в различные позы и набирать очки за наиболее точное повторение позы.

За каждую позу можно набрать от 0 до 100 баллов,
но фактически достичь рзультата в 100 баллов почти невозможно,
т.к. для этого углы наклона рук и ног должны идеально совпадать с эталонными, заложенными в программе.

При запуске игры можно указать количество раундов,
а также в настройках можно менять время,
отведенное на вставание в позу.
## Установка
Программа ожидает, что у вас уже установлен Python 3.12.
Если это не так, то [установите](https://www.python.org/downloads/release/python-31212/) его. Самый простейший способ настнроить программу - запустить `setup.ps1`
для Windows или `setup.sh` для Linux/MacOS.
Это автоматически установит все зависимости и сгенерирует необходимые настройки.
## Зависимости
- Все зависимости прописаны в файле `requirements.txt`
- Если установка производится на Linux, нужно явно установить `pywebview[gtk]` или `pywebview[qt]` в зависимости от используемого графического окружения.
Если вы не знаете, какое у вас окружение, попробуйте сначала установить `pywebview[gtk]`, и, если он не заработает, `pywebview[qt]`.
## Запуск
Для запуска игры достаточно выполнить `start.ps1` на Windows и `start.sh` на Linux/MacOS

Перед запуском нужно **обязательно** выполнить `setup`!
